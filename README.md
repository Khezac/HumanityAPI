# HumanityAPI

Esta é uma API feita para ser consumida pelo e-commerce Humanity.

## Funcionalidades:

- CRUD de produtos em uma API
- Conexão com bucket S3 da AWS para armazenamento de arquivos.

## Tecnologias utilizadas:
- Java
- SpringBoot
- AWS S3
- PostgreSQL
  
* Bibliotecas:
  - AWS SDK Core
  - AWS SDK S3
  - MapStruct
  - ModelMapper
  - OpenApi WebMVC-UI
  - Flyway
  - SpringJPA
